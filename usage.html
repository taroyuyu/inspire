PDF downloading through your university library's subscriptions

It works for journals from Elsevier, Springer and APS. Some universities/institutions, like here in IAS, require you to use a proxy to access paid journals, in which case this feature wouldn't work out of the box. For IAS users you only have to choose "IAS" in the list of the proxies in this application's Preferences. Please contact me if you want to use this feature from such a university.

Bibliography generation for LaTeX
You can make spires.app generate
\begin{thebibliography}...\end{thebibliography}
from your LaTeX file. The only thing you need to do is to put
\cite{Seiberg:1994rs}
etc. in the TeX file. You feed the TeX file to spires.app, and it automatically picks up all the citations and generate \bibitems in a separate file. If you prefer using memorable citation labels such as
\cite{SeibergWitten}
like me, you can specify the mapping in the comment of your tex file, as in
%%map SeibergWitten=Seiberg:1994rs
etc. There are a few extra features controlling spires.app from inside the LaTeX file. See sample.tex for a sample.
